# Merge-Sort-Project

### [16,21,11,8,12,22] -> Merge Sort

## 1- Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.

- [16,21,11] ---- [8,12,22]
- [16,21]  [11] ---- [8,12] [22]
- [16] [21] [11] ---- [8] [12] [22]
- [16,21]  [11] ---- [8,12] [22]
- [11,16,21] ---- [8,12,22]
- [8,11,12,16,21,22]

## 2- Big-O gösterimini yazınız.

- [16,21,11,8,12,22]    ---> n
- [16,21,11] ---- [8,12,22] ---> n/2
- [16,21]  [11] ---- [8,12] [22] ---> n/4
- [16] [21] [11] ---- [8] [12] [22]
- [16,21]  [11] ---- [8,12] [22]
- [11,16,21] ---- [8,12,22]
- [8,11,12,16,21,22]

n+(n/2)+(n/4)+.....n(1+ 1/2+ 1/4....) = n*logn

#### Worst case   : O(n*logn)
#### Average case : O(n*logn)
#### Best case    : O(n*logn)
